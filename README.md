# ukncbtl-qt CMake

it is fork https://github.com/nzeemin/ukncbtl-qt with CMake and some features


### Roadmap

- [x] CMake
- [x] Enable/disable scripting support (UKNCBTL_ENABLE_SCRIPTING default OFF)
- [x] Enable/disable 50Hz framerate (UKNCBTL_ENABLE_50HZ default ON)
- [x] Enable/disable unit testing (UKNCBTL_ENABLE_TESTING default OFF)
- [x] Enable/disable interface languages switch (UKNCBTL_ENABLE_I18N default OFF)
- [x] Enable/disable speed and autosuspend (suspend when focus lost) control menu entries (UKNCBTL_ENABLE_SPEEDCTL default ON)
- [ ] Qt6 migration
- [ ] Audio subsystem 
- [ ] Serial port and network support
